* capf: Short for `completion-at-point-functions` (or the non-plural `completion-at-point-function` depending on context), i.e. a function that provides autocomplete at a given point in the buffer. See [the emacs docs](https://www.gnu.org/software/emacs/manual/html_node/elisp/Completion-in-Buffers.html)
